# @sailshq/upgrade

A tool to help migrate Sails v0.12 apps to Sails version 1.0.

## Usage

#### Install

```bash
npm install @sailshq/upgrade --save-dev
```

#### Upgrade

To upgrade your app:

```bash
sails upgrade
```

and then follow the prompts!
